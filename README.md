## Data Persistence

- All data is automatically saved to your browser's localStorage
- Data persists after closing the browser
- Each date maintains its own independent record
- No internet connection required - works completely offline

## Tips for Long-Term Use

1. **Be Consistent**: Add your daily routine tasks to your timetable
2. **Track Everything**: Log all study sessions to see your progress over time
3. **Review Weekly**: Check the weekly overview to identify patterns
4. **Add Notes**: Use the notes field to document important learnings
5. **Travel Mode**: Mark travel and rest tasks to manage energy effectively

## Technical Details

- **No Framework**: Pure vanilla JavaScript for maximum performance
- **Mobile-First**: Fully responsive design works on all devices
- **Offline-Ready**: No server or internet connection needed
- **Privacy-First**: All data stays in your browser locally
